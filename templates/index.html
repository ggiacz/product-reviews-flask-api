Documentação

Minimal Flask API em Python que hospeda avaliações de produtos em um banco de dados PostgreSQL. Seu principal objetivo é armazenar as notas e os comentários sobre determinados produtos. É necessário uma ferramenta para fornecer enviar comandos http e arquivos JSON ao banco de dados. Recomendo Insomnia, ferramenta que foi utilizada no desenvolvimento. As rotas estão documentadas logo abaixo.

Os próximos passos do desenvolvimento seriam adicionar uma forma de autenticação e vincular os usuários aos seus reviews, além de criar uma interface visual.

Recursos

LÊ A LISTA GERAL DE PRODUTOS: https://product-review-flask-api.onrender.com/api/products

Para muitos dos recursos abaixo é necessário um id de produto. É possível adquirir o id do produto desejado por este recurso. Para utilizá-los, substitua "id-do-produto" pelo valor do id.

ADICIONA PRODUTO: https://product-review-flask-api.onrender.com/api/reviews

Recebe um arquivo JSON e adiciona um produto a lista de produtos. Examplo de entrada: {"name": "Product Name"}

ADICIONA UM REVIEW A UM PRODUTO: https://product-review-flask-api.onrender.com/api/reviews

Recebe um arquivo JSON e adiciona um review a um determinado produto. Examplo de entrada: {"product": 1, "rating": 4.5, "date": "01-01-2021 00:00:00", "feedback": "This is a review."} *A data é opcional

LÊ A QUANTIDADE TOTAL DE REVIEWS: https://product-review-flask-api.onrender.com/api/total-reviews

LÊ AS INFORMAÇÕES DE UM DETERMINADO PRODUTO: https://product-review-flask-api.onrender.com/api/product/id-do-produto

Retorna informações relevantes sobre o produto com o id escolhido.

LÊ LISTA DE REVIEWS DE DETERMINADO PRODUTO: https://product-review-flask-api.onrender.com/api/reviews/id-do-produto

VISUALIZAR LISTA DE PRODUTOS: https://product-review-flask-api.onrender.com/api/visual-products

RENOMEIA PRODUTO: https://product-review-flask-api.onrender.com/api/product/id-do-produto

Recebe um arquivo JSON e renomeia um produto com o id escolhido. Examplo de entrada: {"name": "New Product Name"}

DELETA PRODUTO: https://product-review-flask-api.onrender.com/api/product/id-do-produto

Deleta o produto com o id escolhido.

Tecnologias

O sistema utiliza flask para realizar a funcionalidade da API. ElephantSQL como banco de dados, que é PostgreSQL. Insomnia como ferramenta para testes. Git para versionamento de código. Dotenv para camuflar a senha do banco de dados e passar de forma integrada com gitignore e render.com. O próprio render.com é a plataforma utilizada, juntamente ao github para manter a API no ar.
